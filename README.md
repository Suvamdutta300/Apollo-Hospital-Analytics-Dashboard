# Apollo Hospital Analytics Dashboard

## Project Overview

This Power BI project provides a comprehensive analysis of hospital operations, patient trends, doctor performance, and financial performance for Apollo Hospitals.

The dashboard helps management monitor KPIs, optimize resource utilization, improve patient care, and track revenue performance.

---

## Tools Used

- Power BI
- Excel
- DAX
- Data Modeling
- Data Visualization

---

## Key KPIs

### Executive Overview
- Total Patients: 7.16K
- Total Billing Amount: ₹190.43M
- Average Stay Duration: 8 Days
- Average Feedback Rating: 4.72

### Patient & Doctor Analytics
- Doctor Performance Analysis
- Feedback Rating Distribution
- Test Type Analysis
- Patient Trends

### Financial Analytics
- Total Billing Amount: ₹190.43M
- Insurance Amount: ₹171M
- Net Revenue: ₹19M
- Revenue by Diagnosis
- Monthly Revenue Trend

## Business Insights

- Viral Infection generated the highest revenue.
- Blood Tests were the most frequently performed test.
- Average patient feedback rating remained above 4.7.
- Private beds had the highest occupancy.
- Insurance covered nearly 47% of billing amount.
